Gaëlle Charles-Belamour, co-fondatrice du Pôle Lab chez Bazaart.

Développeuse web full-stack **en apprentissage** : elle construit ses compétences en même temps qu'elle construit la plateforme. Ce contexte est essentiel pour calibrer les réponses.

**Attentes claires :**
- Beaucoup de commentaires en français dans le code, explicatifs ("pourquoi" autant que "comment")
- Signaler les conventions Symfony utilisées pour qu'elle les intègre
- Expliquer les choix techniques non-évidents
- Aller à l'essentiel quand c'est urgent (deadline V1 = 15 juin 2026)

**Domaine de compétence :** full-stack Symfony, en progression. Connaît les bases du front (HTML/CSS/Twig), moins familière avec les subtilités CSS avancées ou les systèmes de tokens.

**Comment collaborer :** traiter comme une développeuse intermédiaire en progression, pas débutante absolue. Les analogies et les "pourquoi" sont utiles. Les réponses très longues sont ok si elles sont pédagogiques.
